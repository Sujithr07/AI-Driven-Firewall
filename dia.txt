Network Interface
    │
    │ (Scapy sniffs every IP packet)
    ▼
extract_features(packet) → 11-feature vector
    │
    ├──→ EnsemblePredictor (RF + XGBoost)
    │        → rf_prediction (0=normal, 1=attack)
    │        → rf_confidence (0.0 - 1.0)
    │
    ├──→ Build RL state tuple (reason, ip_type, protocol, port_type)
    │
    ├──→ DQNAgent.select_action(state)
    │        → "allow" or "block" (ε-greedy)
    │
    ├──→ Compute reward (+1 correct, -1 wrong)
    │        → DQNAgent.update(state, action, reward)
    │
    ├──→ LSTMDetector.add_packet(features, label)
    │        → predict_current() → (is_anomaly, score)
    │        (if anomaly: escalate severity)
    │
    ├──→ ResponseAgent.execute(detection)
    │        → hard_block / rate_limit / quarantine / temp_block
    │        → iptables rule (or dry-run log)
    │
    ├──→ Save to DB (_insert_detection_log) — hash chained
    │
    └──→ FLClient (background thread, every 60s)
             → extract weights → compute delta
             → clip + noise (DP) → POST to FL Server
             → GET global model → apply back
